# ReAct Math Agent

An interactive browser-based ReAct (Reasoning + Acting) agent that solves maths problems step-by-step using a local Ollama LLM. The agent visualises its Thought → Action → Observation reasoning loop in real time, giving full transparency into how it breaks down and solves problems.

🔗 **Live demo:** [brendanjameslynskey.github.io/ReAct_math_agent](https://brendanjameslynskey.github.io/ReAct_math_agent/)

## How It Works

The agent follows the ReAct pattern — a prompting framework where the LLM alternates between reasoning about the problem (Thought), choosing a tool to use (Action), and receiving the result (Observation). This loop repeats until the model reaches a final answer or hits the maximum iteration limit (12 loops).

All computation is performed by deterministic JavaScript tool functions rather than the LLM itself, ensuring numerical accuracy. The LLM's role is purely to reason about *which* computations to perform and in what order.

## Available Tools

- **calculate** — General-purpose expression evaluator supporting arithmetic, exponents, trigonometry, logarithms, factorials, and mathematical constants (π, e)
- **solve_quadratic** — Solves ax² + bx + c = 0, handling real and complex roots
- **factor** — Prime factorisation of positive integers
- **gcd_lcm** — Greatest common divisor and least common multiple of two integers
- **statistics** — Computes mean, median, and standard deviation for a dataset
- **convert_base** — Number base conversion between bases 2–36

## Tech Stack

- **React 18** — UI rendered via Babel standalone (no build step required)
- **Ollama** — Local LLM inference server (default model: Qwen3)
- **Vanilla JS** — Tool implementations with safe expression evaluation
- **GitHub Pages** — Static hosting

## Getting Started

### Prerequisites

1. Install [Ollama](https://ollama.com) and pull a model:
   ```bash
   ollama pull qwen3
   ```

2. Start Ollama with CORS enabled (required for browser access):
   ```bash
   OLLAMA_ORIGINS="*" ollama serve
   ```

### Running Locally

Clone the repo and open `index.html` in a browser — no build tools or dependencies needed:

```bash
git clone https://github.com/BrendanJamesLynskey/ReAct_math_agent.git
cd ReAct_math_agent
open index.html
```

The default Ollama endpoint is `http://localhost:11434`. Both the endpoint URL and model name are configurable in the UI header.
